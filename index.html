Projects & Codes

  1. Data Analysis using Graph Theory

    This an analysis of Douban Movies' Data using graph theory by modelling the movies as nodes of the graph and common actors as edges. The project includes implementation of algorithms as Floyd Algorithm, Prim Algorithm, Dijstra Algorithm, Hungary Algorithm and so on in order to get a variety of data from the graph. I also made a website for interaction using d3.js.

    [Report] [Codes] [Website]

  2. Several Classic Algorithms

    These are small projects for the course Algorithm Design and Analysis including my implementation of nearest point pair algorithm, string matching algorithms, comparison of different sorting algorithms and so on.

    [Report] [Codes]

  3. News Search and Recommendation System

    This is the final project of the course Data Structure. The system includes extracting news from Sina pages, dividing words according to the given dictionary, constructing an index with Red-black Tree, recommendation system using TF-IDF(Term Frequency-Inverse Document Frequency) Algorithm, as well as a GUI based on MFC.

    [Report] [Codes]

    This is my implementation of the labs provided in the book CSAPP(Computer System: A Programmer's Perspective), including Datalab, Buflab, Archlab and Cachelab, which are the projects of the course Principle of the Computer Architecture.
